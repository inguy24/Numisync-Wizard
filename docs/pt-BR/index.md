## Sobre

O NumiSync Wizard conecta seu banco de dados de moedas do OpenNumismat ao abrangente catálogo Numista, identificando automaticamente suas moedas e enriquecendo-as com:

- Informações detalhadas do catálogo (tiragem, composição, governante, gravador)
- Preços de mercado atuais em 4 níveis de conservação (UNC, XF, VF, F)
- Múltiplas variantes de emissão por tipo (ano, marca de cunhagem, tipo)
- Imagens em alta resolução (anverso, reverso e borda)
- Identificação inteligente com busca fuzzy e correção manual

Perfeito para numismatas que querem gastar menos tempo inserindo dados e mais tempo aproveitando sua coleção.

## Funcionalidades

### Identificação Inteligente de Moedas
Busca fuzzy com normalização de denominação, resolução de emissor e suporte a calendários não gregorianos (Meiji, Hijri, Budista Tailandês).

### Controle Granular dos Dados
Escolha exatamente quais dados sincronizar: informações básicas do catálogo, variantes de emissão ou dados de preços. Atualize apenas os campos desejados.

### Comparação Visual de Campos
Comparação lado a lado dos dados existentes versus os novos. Selecione campos individuais para atualizar ou aceite todas as alterações de uma vez.

### Fast Pricing Mode Premium
Atualize os preços de todas as moedas identificadas em segundos. Sem mais precisar clicar moeda por moeda. *(Requer Licença de Apoiador)*

### Auto-Propagate de Dados de Tipo
Aplique automaticamente dados de tipo (tiragem, composição, governante, gravador) a todas as moedas correspondentes da sua coleção. *(Requer Licença de Apoiador)*

### Sincronização Multi-Computador *(Edição Apoiador)*
Compartilhe o cache da API e as configurações entre vários computadores via unidade de rede compartilhada, Dropbox ou NAS.
As contagens de chamadas à API se acumulam corretamente em todas as máquinas. Importação de configurações com um clique em novos computadores.

### Cache Inteligente
Cache persistente da API reduz requisições redundantes e respeita os limites de taxa do Numista. Funciona offline com dados em cache.

### Mapeamento Avançado de Campos
Personalize como mais de 45 fontes de dados do Numista se mapeiam para os campos do OpenNumismat. Controle total sobre seus dados.

## Início Rápido

Comece em 5 passos simples:

1. **Instale o NumiSync Wizard** e abra o aplicativo
2. **Abra sua coleção do OpenNumismat** (arquivo .db)
3. **Settings → Adicione sua chave de API do Numista** (gratuita em [numista.com](https://www.numista.com/))
4. **Selecione moedas → Clique em "Search & Enrich"** para encontrar correspondências
5. **Revise as correspondências → Aceite as alterações → Pronto!**

Para instruções detalhadas, consulte o [Guia de Instalação](/pt-BR/installation) e o [Guia de Início Rápido](/pt-BR/quickstart).

**Suporte de Plataformas:**
- **Windows 10/11** - Suporte completo com instalador
- **macOS 10.13+** - Intel e Apple Silicon (M1/M2/M3)
- **Linux** - Debian/Ubuntu (.deb), Fedora/RHEL (.rpm), Universal (AppImage)

## Softwares Necessários

### OpenNumismat
**[Baixar OpenNumismat](https://opennumismat.github.io/)** - Software gratuito e de código aberto para gerenciamento de coleções de moedas para Windows, macOS e Linux. O OpenNumismat oferece um banco de dados poderoso para catalogar suas moedas, e o NumiSync Wizard o aprimora preenchendo automaticamente os dados do catálogo e os preços.

### Numista
**[Visitar Numista.com](https://www.numista.com/)** - O maior catálogo numismático online do mundo, com informações detalhadas sobre moedas de todo o globo. O NumiSync Wizard usa a API do Numista para buscar dados de catálogo, preços e imagens. Você precisará de uma conta gratuita no Numista para obter uma chave de API.
