## 啟用授權

### 第一步：複製您的授權金鑰

購買後，您將收到來自 Polar.sh 的電子郵件，其中包含您的授權金鑰，格式如下：

```
V1-XXXX-XXXX-XXXX-XXXX
```

### 第二步：開啟 NumiSync Wizard

在您的電腦上啟動 NumiSync Wizard 應用程式。

### 第三步：輸入授權金鑰

1. 點擊選單列中的 **Settings**（或按 `Ctrl+,` / `Cmd+,`）
2. 導覽至 **License** 索引標籤
3. 將授權金鑰貼上至文字欄位
4. 點擊 **Activate**

### 第四步：驗證啟用

啟用成功後，您將看到：
- ✅ 授權狀態：**Active**
- 進階功能立即解鎖
- 不再出現提示彈出視窗
