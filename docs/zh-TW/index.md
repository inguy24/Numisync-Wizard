## 關於

NumiSync Wizard 將您的 OpenNumismat 錢幣收藏資料庫與完整的 Numista 目錄相連結，自動比對您的錢幣並為其補充以下資訊：

- 詳細的目錄資訊（鑄造量、成分、統治者、設計師）
- 四個品相等級的當前市場價格（UNC、XF、VF、F）
- 每種類型的多個版別（年份、鑄幣標記、類型）
- 高解析度圖片（正面、反面及邊緣）
- 智慧比對，支援模糊搜尋與手動覆蓋

專為希望減少資料輸入、將更多時間用於享受收藏樂趣的錢幣愛好者打造。

## 功能特色

### 智慧錢幣比對
支援面額規範化、發行方解析及非西曆紀年（明治、伊斯蘭曆、泰國佛曆）的模糊搜尋。

### 精細的資料控制
精確選擇需要同步的資料類型：基本目錄資訊、版別資訊或定價資料。只更新您想要的欄位。

### 視覺化欄位比較
並排比較現有資料與新資料。可單獨選取欄位更新，也可一次接受所有變更。

### 進階快速定價模式
批次更新所有已比對錢幣的價格，無需逐枚點擊。*（需要支持者授權）*

### 自動傳播類型資料
自動將類型級資料（鑄造量、成分、統治者、設計師）套用至收藏中所有符合的錢幣。*（需要支持者授權）*

### 多機同步 *（支持者版）*
透過共享網路磁碟、Dropbox 或 NAS 在多台電腦之間共享 API 快取和設定。
API 呼叫次數在所有裝置上正確累計。新裝置一鍵匯入設定。

### 智慧快取
持久化 API 快取減少重複請求，遵守 Numista 速率限制。可使用快取資料離線工作。

### 進階欄位對應
自訂 Numista 45+ 個資料來源如何對應至 OpenNumismat 欄位，完全掌控您的資料。

## 快速開始

五個簡單步驟即可上手：

1. **安裝 NumiSync Wizard** 並啟動應用程式
2. **開啟您的 OpenNumismat 收藏**（.db 檔案）
3. **設定 → 新增您的 Numista API 金鑰**（從 [numista.com](https://www.numista.com/) 免費取得）
4. **選取錢幣 → 點擊「Search & Enrich」** 尋找比對項目
5. **查看比對結果 → 接受變更 → 完成！**

詳細說明請參閱[安裝指南](/zh-TW/installation)和[快速入門指南](/zh-TW/quickstart)。

**平台支援：**
- **Windows 10/11** - 完整支援，提供安裝程式
- **macOS 10.13+** - 支援 Intel 及 Apple Silicon（M1/M2/M3）
- **Linux** - Debian/Ubuntu（.deb）、Fedora/RHEL（.rpm）、通用版（AppImage）

## 必要軟體

### OpenNumismat
**[下載 OpenNumismat](https://opennumismat.github.io/)** - 適用於 Windows、macOS 及 Linux 的免費開源錢幣收藏管理軟體。OpenNumismat 提供強大的資料庫用於整理您的錢幣，而 NumiSync Wizard 透過自動填入目錄資料和定價資訊來增強它。

### Numista
**[造訪 Numista.com](https://www.numista.com/)** - 全球最大的線上錢幣目錄，收錄了來自世界各地的詳細錢幣資訊。NumiSync Wizard 使用 Numista API 取得目錄資料、定價和圖片。您需要一個免費的 Numista 帳號來取得 API 金鑰。
