## I-activate ang Iyong License

### Hakbang 1: Kopyahin ang Iyong License Key

Pagkatapos ng pagbili, makakatanggap ka ng email mula sa Polar.sh na may iyong license key. Magiging ganito ang hitsura nito:

```
V1-XXXX-XXXX-XXXX-XXXX
```

### Hakbang 2: Buksan ang NumiSync Wizard

Ilunsad ang aplikasyong NumiSync Wizard sa iyong computer.

### Hakbang 3: Ilagay ang License Key

1. I-click ang **Settings** sa menu bar (o pindutin ang `Ctrl+,` / `Cmd+,`)
2. Pumunta sa tab na **License**
3. I-paste ang iyong license key sa text field
4. I-click ang **Activate**

### Hakbang 4: I-verify ang Activation

Kapag na-activate na, makikita mo ang:
- Status ng license: **Active**
- Agarang nai-unlock ang mga premium na tampok
- Walang mga nag prompt na

## Mga Madalas Itanong

### Maaari ko bang gamitin ang aking license sa maraming computer?

Oo! Maaari mong i-activate ang iyong license sa hanggang **5 device**. Perpekto para sa home, trabaho, laptop, at backup na mga makina.

### Ito ba ay isang subscription?

Hindi! **Bayad nang isang beses, gamitin magpakailanman.** Ang iyong license ay hindi kailanman mag-e-expire at kasama ang lahat ng tampok mula sa bersyon 1.x.

### Paano ang mga susunod na major na bersyon (v2.0, v3.0)?

Ang iyong v1 license ay nag-i-unlock ng lahat ng tampok ng v1.x nang permanente. Ang mga susunod na major na bersyon (v2.0+) na may makabuluhang bagong tampok ay maaaring mangailangan ng hiwalay na license, ngunit ang mga kasalukuyang tampok ng v1 ay patuloy na gagana. Ang mga maagang supporter ay makakatanggap ng mga diskwento sa pag-upgrade.

### Nawala ko ang aking license key. Maaari ko ba itong makuha?

Oo! Ang iyong license key ay nakatali sa iyong email address. Suriin ang iyong email ng kumpirmasyon ng pagbili mula sa Polar.sh, o makipag-ugnayan sa suporta gamit ang iyong email ng pagbili.

### Sinusuportahan ba nito ang developer?

Tiyak! Ang iyong $10 ay tumutulong sa pagsaklaw ng:
- Pag-host ng domain at mga SSL certificate
- Imprastraktura ng API at mga server ng cache
- Oras ng pagpapaunlad para sa mga bagong tampok
- Mga sertipiko ng code signing (kapag available)
- Kape para sa mga gabi na debugging session

Ang bawat supporter license ay direktang nagpopondo sa patuloy na pagpapaunlad ng NumiSync at pinapanatiling libre ang pangunahing paggamit.
