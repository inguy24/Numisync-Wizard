## Tungkol Sa

Ikinokonekta ng NumiSync Wizard ang iyong database ng koleksyon ng barya sa OpenNumismat sa komprehensibong katalogo ng Numista, awtomatikong tinutugma ang iyong mga barya at pinayaman ang mga ito ng:

- Detalyadong impormasyon ng katalogo (mintage, komposisyon, pinuno, taga-disenyo)
- Kasalukuyang presyo ng merkado sa 4 antas ng grado (UNC, XF, VF, F)
- Maramihang variant ng isyu bawat uri (taon, mint mark, uri)
- Mataas na resolusyong mga larawan (harap, likod, at gilid)
- Matalinong pagtutugma gamit ang fuzzy search at manu-manong pagwawasto

Perpekto para sa mga numismatiko na gustong gumugol ng mas kaunting oras sa pagpasok ng data at mas maraming oras sa pag-enjoy ng kanilang koleksyon.

## Mga Tampok

### Matalinong Pagtutugma ng Barya
Fuzzy search na may normalisasyon ng denominasyon, resolusyon ng issuer, at suporta sa mga kalendaryo na hindi Gregoriano (Meiji, Hijri, Thai Buddhist).

### Granular na Kontrol ng Data
Piliin kung anong data ang isi-sync: pangunahing impormasyon ng katalogo, mga variant ng isyu, o data ng presyo. I-update lamang ang mga field na gusto mo.

### Visual na Paghahambing ng Field
Paghahambing ng kasalukuyang data kumpara sa bagong data. Pumili ng mga indibidwal na field para i-update o tanggapin ang lahat ng pagbabago nang sabay-sabay.

### Premium Fast Pricing Mode
I-batch update ang presyo ng lahat ng naitugmang barya sa loob ng ilang segundo. Hindi na kailangang mag-click ng isa-isa sa bawat barya. *(Nangangailangan ng Supporter License)*

### Auto-Propagate ng Data ng Uri
Awtomatikong ilapat ang data ng uri (mintage, komposisyon, pinuno, taga-disenyo) sa lahat ng katugmang barya sa iyong koleksyon. *(Nangangailangan ng Supporter License)*

### Multi-Machine Sync *(Supporter Edition)*
Ibahagi ang API cache at mga setting sa maraming computer sa pamamagitan ng shared network drive, Dropbox, o NAS.
Naiiipon nang tama ang bilang ng API call sa lahat ng makina. One-click na pag-import ng mga setting sa mga bagong makina.

### Matalinong Caching
Ang persistent na API cache ay nagbabawas ng mga paulit-ulit na kahilingan at gumagalang sa mga limitasyon ng rate ng Numista. Gumagana nang offline gamit ang naka-cache na data.

### Advanced na Pagmamapa ng Field
I-customize kung paano nag-map ang mahigit 45 pinagkukunan ng data ng Numista sa mga field ng OpenNumismat. Ganap na kontrol sa iyong data.

## Mabilis na Pagsisimula

Magsimula sa 5 simpleng hakbang:

1. **I-install ang NumiSync Wizard** at ilunsad ang aplikasyon
2. **Buksan ang iyong koleksyon ng OpenNumismat** (file na .db)
3. **Settings → Idagdag ang iyong Numista API key** (libre mula sa [numista.com](https://www.numista.com/))
4. **Pumili ng mga barya → I-click ang "Search & Enrich"** upang mahanap ang mga tugma
5. **Suriin ang mga tugma → Tanggapin ang mga pagbabago → Tapos na!**

Para sa mas detalyadong mga tagubilin, tingnan ang [Gabay sa Pag-install](/tl/installation) at ang [Gabay sa Mabilis na Pagsisimula](/tl/quickstart).

**Suporta ng Platform:**
- **Windows 10/11** - Buong suporta na may installer
- **macOS 10.13+** - Intel at Apple Silicon (M1/M2/M3)
- **Linux** - Debian/Ubuntu (.deb), Fedora/RHEL (.rpm), Universal (AppImage)

## Kinakailangang Software

### OpenNumismat
**[I-download ang OpenNumismat](https://opennumismat.github.io/)** - Libreng open-source na software para sa pamamahala ng koleksyon ng barya para sa Windows, macOS, at Linux. Nagbibigay ang OpenNumismat ng makapangyarihang database para sa pagkatalong ng iyong mga barya, at pinipigilan ito ng NumiSync Wizard sa pamamagitan ng awtomatikong pagpuno ng data ng katalogo at presyo.

### Numista
**[Bisitahin ang Numista.com](https://www.numista.com/)** - Ang pinakamalaking online na numismatic na katalogo sa mundo na may detalyadong impormasyon sa mga barya mula sa buong mundo. Ginagamit ng NumiSync Wizard ang Numista API upang makuha ang data ng katalogo, presyo, at mga larawan. Kakailanganin mo ng libreng account sa Numista upang makakuha ng API key.
