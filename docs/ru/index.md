## О программе

NumiSync Wizard связывает вашу базу данных коллекции монет OpenNumismat с обширным каталогом Numista, автоматически находя совпадения и обогащая ваши монеты:

- Подробной информацией из каталога (тираж, состав, правитель, дизайнер)
- Актуальными рыночными ценами по 4 уровням состояния (UNC, XF, VF, F)
- Несколькими вариантами выпуска каждого типа (год, знак монетного двора, разновидность)
- Высококачественными изображениями (аверс, реверс и гурт)
- Интеллектуальным поиском с нечётким соответствием и ручным управлением

Идеальное решение для нумизматов, которые хотят тратить меньше времени на ввод данных и больше — на коллекционирование.

## Возможности

### Интеллектуальный поиск монет
Нечёткий поиск с нормализацией номиналов, определением эмитентов и поддержкой негрегорианских систем летосчисления (японский, исламский хиджри, тайский буддийский календари).

### Детальный контроль над данными
Выбирайте, какие именно данные синхронизировать: основную информацию из каталога, варианты выпусков или ценовые данные. Обновляйте только нужные поля.

### Визуальное сравнение полей
Сравнение старых и новых данных рядом. Выбирайте отдельные поля для обновления или принимайте все изменения сразу.

### Быстрое обновление цен (Premium)
Массовое обновление цен для всех подобранных монет за секунды. Больше не нужно кликать по каждой монете отдельно. *(Требуется лицензия Supporter)*

### Авто-распространение данных типа
Автоматическое применение данных уровня типа (тираж, состав, правитель, дизайнер) ко всем соответствующим монетам в вашей коллекции. *(Требуется лицензия Supporter)*

### Синхронизация между устройствами *(Supporter Edition)*
Делитесь кэшем API и настройками между несколькими компьютерами через общий сетевой диск, Dropbox или NAS.
Счётчик вызовов API корректно суммируется по всем машинам. Импорт настроек на новые компьютеры в один клик.

### Умное кэширование
Постоянный кэш API снижает количество запросов и соблюдает ограничения Numista. Работает в автономном режиме с кэшированными данными.

### Расширенное сопоставление полей
Настраивайте соответствие между 45+ источниками данных Numista и полями OpenNumismat. Полный контроль над вашими данными.

## Быстрый старт

Начните работу за 5 простых шагов:

1. **Установите NumiSync Wizard** и запустите приложение
2. **Откройте вашу коллекцию OpenNumismat** (файл .db)
3. **Settings → Введите ваш API-ключ Numista** (бесплатно на [numista.com](https://www.numista.com/))
4. **Выберите монеты → Нажмите "Search & Enrich"** для поиска совпадений
5. **Просмотрите совпадения → Принимайте изменения → Готово!**

Подробные инструкции смотрите в [Руководстве по установке](/ru/installation) и [Руководстве по быстрому старту](/ru/quickstart).

**Поддерживаемые платформы:**
- **Windows 10/11** — Полная поддержка с установщиком
- **macOS 10.13+** — Intel и Apple Silicon (M1/M2/M3)
- **Linux** — Debian/Ubuntu (.deb), Fedora/RHEL (.rpm), Универсальный (AppImage)

## Необходимое программное обеспечение

### OpenNumismat
**[Скачать OpenNumismat](https://opennumismat.github.io/)** — Бесплатная программа с открытым исходным кодом для управления коллекцией монет (Windows, macOS, Linux). OpenNumismat предоставляет мощную базу данных для каталогизации монет, а NumiSync Wizard расширяет её возможности, автоматически заполняя данные каталога и цены.

### Numista
**[Посетить Numista.com](https://www.numista.com/)** — Крупнейший в мире онлайн-каталог монет с подробной информацией о монетах со всего мира. NumiSync Wizard использует API Numista для получения каталожных данных, цен и изображений. Для получения API-ключа необходима бесплатная учётная запись Numista.
