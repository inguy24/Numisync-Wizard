# Руководство по быстрому старту

Начните работу с NumiSync Wizard за 5 минут. Это руководство проведёт вас через основной процесс обогащения вашей коллекции монет.

**Примечание о платформах:** Данное руководство подходит для Windows, macOS и Linux. Горячие клавиши указаны для всех платформ там, где они различаются.

---

## Предварительные условия

Перед началом убедитесь, что у вас есть:

- **Установленный NumiSync Wizard** ([Руководство по установке](/ru/installation))
- **Коллекция OpenNumismat** (файл .db с монетами)
- **API-ключ Numista** (бесплатно на [numista.com](https://www.numista.com/))

---

## Шаг 1: Запуск и настройка

### Откройте NumiSync Wizard

1. Запустите NumiSync Wizard:
   - **Windows:** Меню «Пуск» или ярлык на рабочем столе
   - **macOS:** Папка Applications или Launchpad
   - **Linux:** Меню приложений или выполните `numisync-wizard` (если установлено через .deb/.rpm)
2. При первом запуске папка кэша будет создана автоматически

### Добавьте API-ключ

1. Нажмите **Settings** (значок шестерёнки) или нажмите:
   - **Windows/Linux:** `Ctrl+,`
   - **macOS:** `Cmd+,`
2. Перейдите на вкладку **API Settings**
3. Вставьте ваш API-ключ Numista
4. Нажмите **Save**

**Нет API-ключа?** Получите бесплатно на [numista.com](https://www.numista.com/) → Профиль → API Access

---

## Шаг 2: Откройте вашу коллекцию

1. Нажмите **File → Open Collection** или нажмите:
   - **Windows/Linux:** `Ctrl+O`
   - **macOS:** `Cmd+O`
2. Перейдите к вашему файлу OpenNumismat `.db`
3. Нажмите **Open**
4. Ваши монеты загрузятся в главном окне

**Совет:** NumiSync запоминает последние коллекции. Используйте **File → Recent Collections** для быстрого доступа.

---

## Шаг 3: Поиск совпадений

### Выберите монеты для обогащения

Вы можете обогащать монеты по одной или группами:

- **Одна монета:** Нажмите на строку монеты для выбора
- **Несколько монет:** Удерживайте клавишу-модификатор и нажимайте на строки
  - **Windows/Linux:** `Ctrl+Click`
  - **macOS:** `Cmd+Click`
- **Диапазон:** Нажмите на первую монету, удерживайте `Shift`, нажмите на последнюю
- **Все монеты:** Выбрать все
  - **Windows/Linux:** `Ctrl+A`
  - **macOS:** `Cmd+A`

### Запустите поиск

1. Нажмите кнопку **Search & Enrich** (или нажмите `F2`)
2. NumiSync выполнит поиск в Numista для каждой выбранной монеты
3. Индикатор прогресса показывает текущий статус

**Что происходит:**
- Поиск по номиналу, стране, году, знаку монетного двора
- Обработка вариаций (например, «Cent» и «Cents», «USA» и «United States»)
- Поддержка негрегорианских календарей (годы Мэйдзи, хиджри и др.)
- Использование кэшированных результатов при наличии (быстрее!)

---

## Шаг 4: Просмотр совпадений

### Понимание результатов поиска

После поиска каждая монета отображает один из трёх статусов:

- **Match Found** — Запись в каталоге Numista найдена
- **Multiple Matches** — Несколько вариантов (требуется ручной выбор)
- **No Match** — Запись не найдена (попробуйте ручной поиск)

### Просмотр сравнения полей

1. Нажмите на монету с найденным совпадением
2. **Панель Field Comparison** показывает:
   - **Левый столбец:** Ваши существующие данные
   - **Правый столбец:** Данные из каталога Numista
   - **Различия выделены** цветом
3. Просмотрите, что изменится

---

## Шаг 5: Принятие или уточнение совпадений

### Принять все изменения

Если совпадение выглядит верным:
1. Нажмите кнопку **Accept Match** (или нажмите `Enter`)
2. Все данные Numista немедленно обновятся в вашей монете
3. Монета отмечается как обогащённая

### Выборочное обновление полей

Чтобы обновить только определённые поля:
1. В панели Field Comparison **снимите флажки** с полей, которые не нужно обновлять
2. Нажмите **Accept Match**
3. Будут обновлены только отмеченные поля

### Выбор другого выпуска

Многие монеты имеют несколько выпусков (годы, знаки двора, разновидности):

1. Нажмите кнопку **Choose Issue**
2. **Диалог Issue Picker** показывает все варианты
3. Выберите правильный выпуск для вашей монеты
4. Сравнение полей обновится с данными этого выпуска
5. Нажмите **Accept Match**

### Ручной поиск

Если совпадение не найдено автоматически:
1. Нажмите кнопку **Manual Search** или нажмите:
   - **Windows/Linux:** `Ctrl+F`
   - **macOS:** `Cmd+F`
2. Измените параметры поиска (номинал, год, страна)
3. Нажмите **Search**
4. Просмотрите результаты и выберите правильную запись
5. Нажмите **Accept Match**

---

## Шаг 6: Загрузка изображений (необязательно)

### Автоматическая загрузка изображений

Если **Data Settings → Images** включён:
- Изображения загружаются автоматически при принятии совпадения
- Изображения аверса, реверса и гурта (если доступны)
- Сохраняются в папке изображений OpenNumismat

### Ручная загрузка изображений

1. Выберите обогащённую монету
2. Нажмите кнопку **Download Images**
3. Выберите, какие изображения загрузить (аверс, реверс, гурт)
4. Нажмите **Download**

**Совет:** Используйте **Image Comparison** для предварительного просмотра перед принятием

## Типовые сценарии работы

### Сценарий 1: Обогащение новой коллекции

1. Откройте коллекцию с большим количеством необогащённых монет
2. Выберите все монеты (`Ctrl+A`)
3. Нажмите **Search & Enrich** (или нажмите `F2`)
4. Просматривайте совпадения одно за другим
5. Принимайте совпадения по ходу работы
6. Используйте ручной поиск для монет без совпадений

**Экономия времени:** 2–3 минуты на монету → 10–15 секунд на монету

### Сценарий 2: Обновление только цен

1. Перейдите в **Settings → Data Settings**
2. Снимите флажки **Basic** и **Issue** (оставьте **Pricing** отмеченным)
3. Выберите монеты для обновления
4. Нажмите **Search & Enrich**
5. Принимайте совпадения (обновляются только цены)

**Совет профи:** Получите [лицензию Supporter](#) для использования **Fast Pricing Mode** — обновляет все подобранные монеты мгновенно!

### Сценарий 3: Исправление неверных совпадений

1. Выберите монету с неверными данными
2. Нажмите **Manual Search**
3. Найдите правильную запись каталога
4. Примите совпадение
5. Старые данные будут перезаписаны

**Совет:** Используйте **Field Comparison** для проверки перед принятием

## Советы для лучших результатов

### Советы по поиску

**Рекомендации:**
- Начинайте с монет, имеющих полную информацию (год, страна, номинал)
- Используйте стандартные сокращения номиналов («1 Cent», а не «1c»)
- Позвольте NumiSync автоматически нормализовать номиналы

**Избегайте:**
- Поиска монет без ключевых полей (страна, номинал)
- Ручного редактирования поисковых запросов без необходимости
- Не проверяйте первое совпадение — всегда убеждайтесь в его правильности!

### Качество данных

**Рекомендации:**
- Просматривайте Field Comparison перед принятием
- Используйте Issue Picker при наличии нескольких вариантов
- Проверяйте, что изображения соответствуют вашей физической монете

**Избегайте:**
- Слепого принятия всех совпадений
- Перезаписи хороших данных неполными данными каталога
- Не забывайте сначала сделать резервную копию коллекции!

### Производительность

**Рекомендации:**
- Включите кэширование (Settings → General → Cache)
- Работайте партиями по 10–20 монет
- Используйте Fast Pricing Mode для больших обновлений (лицензия Supporter)

**Избегайте:**
- Одновременного поиска 1000+ монет (соблюдаются ограничения, но это медленно)
- Отключения кэширования (расходует API-запросы впустую)
- Повторного поиска одной монеты (используйте кэш)

**Windows/Linux:**
- `Ctrl+O` — Открыть коллекцию
- `F2` — Search & Enrich выбранных монет

**macOS:**
- `Cmd+O` — Открыть коллекцию
- `F2` — Search & Enrich выбранных монет
- `Cmd+F` — Ручной поиск
- `Enter` — Принять совпадение
- `Escape` — Отмена/Закрыть диалог
- `Cmd+A` — Выбрать все монеты
- `Cmd+,` — Открыть настройки

## Что дальше?

### Изучите премиум-функции

Получите **[лицензию Supporter ($10)](#)** для разблокировки:
- **Fast Pricing Mode** — Массовое обновление цен для всех подобранных монет
- **Auto-Propagate** — Автоматическое применение данных типа к соответствующим монетам
- **Никаких надоедливых напоминаний!**

### Расширенные функции

- **Field Mapping** — Настройте соответствие данных Numista вашим полям
- **Batch Operations** — Эффективная обработка сотен монет
- **Multi-Machine Support** — Общий кэш на всех устройствах
- **Custom Cache Location** — Хранение кэша на сетевом диске

## Нужна помощь?

### Часто задаваемые вопросы

**В: Почему моя монета не нашла совпадения?**
- О: Возможно, стране или номиналу требуется нормализация. Попробуйте ручной поиск с вариациями.

**В: Почему некоторые поля не обновляются?**
- О: Проверьте **Data Settings** — некоторые категории данных могут быть отключены.

**В: Можно ли отменить принятое совпадение?**
- О: Автоматически нет. Восстановите из резервной копии или вручную верните данные.

**В: Как обновить только цены без изменения других полей?**
- О: Settings → Data Settings → Снимите флажки Basic и Issue, оставьте Pricing отмеченным.

**В: Что происходит при повторном поиске монеты?**
- О: NumiSync использует кэшированные результаты (мгновенно), если вы не нажмёте «Refresh from API».
