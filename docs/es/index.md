## Acerca de

NumiSync Wizard conecta tu base de datos de colección de monedas de OpenNumismat con el exhaustivo catálogo de Numista, encontrando automáticamente tus monedas y enriqueciéndolas con:

- Información detallada del catálogo (tirada, composición, gobernante, diseñador)
- Precios de mercado actuales en 4 niveles de conservación (UNC, XF, VF, F)
- Múltiples variantes de emisión por tipo (año, marca de ceca, tipo)
- Imágenes en alta resolución (anverso, reverso y canto)
- Identificación inteligente con búsqueda difusa y corrección manual

Ideal para numismáticos que quieren dedicar menos tiempo a introducir datos y más tiempo a disfrutar de su colección.

## Características

### Identificación Inteligente de Monedas
Búsqueda difusa con normalización de denominaciones, resolución de emisores y soporte para calendarios no gregorianos (Meiji, Hijri, calendarios budistas tailandeses).

### Control Granular de Datos
Elige exactamente qué datos sincronizar: información básica del catálogo, variantes de emisión o datos de precios. Actualiza solo los campos que desees.

### Comparación Visual de Campos
Comparación lado a lado de los datos existentes frente a los nuevos. Selecciona campos individuales para actualizar o acepta todos los cambios a la vez.

### Fast Pricing Mode Premium
Actualización masiva de precios para todas las monedas identificadas en segundos. Sin más clics moneda por moneda. *(Requiere Licencia de Supporter)*

### Auto-Propagate de Datos de Tipo
Aplica automáticamente los datos a nivel de tipo (tirada, composición, gobernante, diseñador) a todas las monedas coincidentes de tu colección. *(Requiere Licencia de Supporter)*

### Sincronización Multimáquina *(Edición Supporter)*
Comparte la caché de la API y la configuración entre varios ordenadores mediante una unidad de red compartida, Dropbox o NAS.
Los recuentos de llamadas a la API se acumulan correctamente en todas las máquinas. Importación de configuración con un clic en máquinas nuevas.

### Caché Inteligente
La caché persistente de la API reduce las solicitudes redundantes y respeta los límites de velocidad de Numista. Funciona sin conexión con los datos en caché.

### Mapeo Avanzado de Campos
Personaliza cómo más de 45 fuentes de datos de Numista se asignan a los campos de OpenNumismat. Control total sobre tus datos.

## Inicio Rápido

Empieza a funcionar en 5 sencillos pasos:

1. **Instala NumiSync Wizard** y lanza la aplicación
2. **Abre tu colección de OpenNumismat** (archivo .db)
3. **Settings → Añade tu clave de API de Numista** (gratuita en [numista.com](https://www.numista.com/))
4. **Selecciona monedas → Haz clic en "Search & Enrich"** para encontrar coincidencias
5. **Revisa las coincidencias → Acepta los cambios → ¡Listo!**

Para instrucciones detalladas, consulta la [Guía de Instalación](/es/installation) y la [Guía de Inicio Rápido](/es/quickstart).

**Compatibilidad de plataformas:**
- **Windows 10/11** - Soporte completo con instalador
- **macOS 10.13+** - Intel y Apple Silicon (M1/M2/M3)
- **Linux** - Debian/Ubuntu (.deb), Fedora/RHEL (.rpm), Universal (AppImage)

## Software Necesario

### OpenNumismat
**[Descargar OpenNumismat](https://opennumismat.github.io/)** - Software gratuito y de código abierto para gestionar colecciones de monedas, disponible para Windows, macOS y Linux. OpenNumismat ofrece una potente base de datos para catalogar tus monedas, y NumiSync Wizard la complementa rellenando automáticamente los datos del catálogo y los precios.

### Numista
**[Visita Numista.com](https://www.numista.com/)** - El catálogo numismático en línea más grande del mundo, con información detallada sobre monedas de todo el mundo. NumiSync Wizard utiliza la API de Numista para obtener datos del catálogo, precios e imágenes. Necesitarás una cuenta gratuita en Numista para obtener una clave de API.
