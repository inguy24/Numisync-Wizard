## Attiva la Tua Licenza

### Passaggio 1: Copia la Tua Chiave di Licenza

Dopo l'acquisto, riceverai un'e-mail da Polar.sh con la tua chiave di licenza. Avrà questo aspetto:

```
V1-XXXX-XXXX-XXXX-XXXX
```

### Passaggio 2: Apri NumiSync Wizard

Avvia l'applicazione NumiSync Wizard sul tuo computer.

### Passaggio 3: Inserisci la Chiave di Licenza

1. Clicca su **Settings** nella barra dei menu (o premi `Ctrl+,` / `Cmd+,`)
2. Vai alla scheda **License**
3. Incolla la tua chiave di licenza nel campo di testo
4. Clicca su **Activate**

### Passaggio 4: Verifica l'Attivazione

Una volta attivata, vedrai:
- Stato della licenza: **Active**
- Funzioni premium sbloccate immediatamente
- Niente più notifiche fastidiose

## Domande Frequenti

### Posso usare la mia licenza su più computer?

Sì! Puoi attivare la tua licenza su un massimo di **5 dispositivi**. Perfetta per casa, lavoro, portatile e macchine di backup.

### È un abbonamento?

No! **Acquisto unico, uso per sempre.** La tua licenza non scade e include tutte le funzioni della versione 1.x.

### Cosa succede con le future versioni principali (v2.0, v3.0)?

La tua licenza v1 sblocca tutte le funzioni della versione 1.x in modo permanente. Le future versioni principali (v2.0+) con funzioni nuove significative potrebbero richiedere una licenza separata, ma le funzioni attuali della v1 continueranno a funzionare. I supporter attuali riceveranno sconti di aggiornamento.

### Ho perso la mia chiave di licenza. Posso recuperarla?

Sì! La tua chiave di licenza è collegata al tuo indirizzo e-mail. Controlla l'e-mail di conferma dell'acquisto da Polar.sh, oppure contatta il supporto con l'e-mail dell'acquisto.

### Questo supporta lo sviluppatore?

Assolutamente! I tuoi $10 aiutano a coprire:
- Hosting del dominio e certificati SSL
- Infrastruttura API e server di cache
- Tempo di sviluppo per nuove funzioni
- Certificati di firma del codice (quando disponibili)
- Caffè per le sessioni di debug notturne

Ogni licenza supporter finanzia direttamente lo sviluppo continuo di NumiSync e lo mantiene gratuito per l'uso di base.
