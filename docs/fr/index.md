## À propos

NumiSync Wizard connecte votre base de données de collection de monnaies OpenNumismat au catalogue complet Numista, en identifiant automatiquement vos pièces et en les enrichissant avec :

- Des informations détaillées du catalogue (tirage, composition, souverain, graveur)
- Les prix actuels du marché sur 4 niveaux de conservation (UNC, XF, VF, F)
- Plusieurs variantes d'émission par type (année, marque d'atelier, type)
- Des images haute résolution (avers, revers et tranche)
- Une correspondance intelligente avec recherche floue et correction manuelle

Idéal pour les numismates qui souhaitent passer moins de temps à saisir des données et plus de temps à profiter de leur collection.

## Fonctionnalités

### Identification intelligente des pièces
Recherche floue avec normalisation des dénominations, résolution des émetteurs et prise en charge des calendriers non grégoriens (Meiji, Hijri, bouddhiste thaïlandais).

### Contrôle granulaire des données
Choisissez précisément les données à synchroniser : informations de base du catalogue, variantes d'émission ou données de prix. Ne mettez à jour que les champs souhaités.

### Comparaison visuelle des champs
Comparaison côte à côte des données existantes et des nouvelles données. Sélectionnez les champs individuels à mettre à jour ou acceptez toutes les modifications en une seule fois.

### Mode tarification rapide Premium
Mettez à jour les prix en masse pour toutes les pièces identifiées en quelques secondes. Fini de cliquer sur chaque pièce une par une. *(Nécessite une licence Supporter)*

### Propagation automatique des données de type
Appliquez automatiquement les données de type (tirage, composition, souverain, graveur) à toutes les pièces correspondantes de votre collection. *(Nécessite une licence Supporter)*

### Synchronisation multi-machines *(Édition Supporter)*
Partagez le cache API et les paramètres sur plusieurs ordinateurs via un lecteur réseau partagé, Dropbox ou NAS.
Les compteurs d'appels API s'accumulent correctement sur toutes les machines. Importation des paramètres en un clic sur les nouveaux appareils.

### Mise en cache intelligente
Le cache API persistant réduit les requêtes redondantes et respecte les limites de débit de Numista. Fonctionne hors ligne avec les données en cache.

### Mappage avancé des champs
Personnalisez la façon dont 45+ sources de données Numista se mappent aux champs OpenNumismat. Contrôle total sur vos données.

## Démarrage rapide

Lancez-vous en 5 étapes simples :

1. **Installez NumiSync Wizard** et lancez l'application
2. **Ouvrez votre collection OpenNumismat** (fichier .db)
3. **Settings → Ajoutez votre clé API Numista** (gratuite sur [numista.com](https://www.numista.com/))
4. **Sélectionnez des pièces → Cliquez sur "Search & Enrich"** pour trouver des correspondances
5. **Examinez les correspondances → Acceptez les modifications → C'est fait !**

Pour des instructions détaillées, consultez le [Guide d'installation](/fr/installation) et le [Guide de démarrage rapide](/fr/quickstart).

**Compatibilité des plateformes :**
- **Windows 10/11** - Prise en charge complète avec installateur
- **macOS 10.13+** - Intel et Apple Silicon (M1/M2/M3)
- **Linux** - Debian/Ubuntu (.deb), Fedora/RHEL (.rpm), Universel (AppImage)

## Logiciels requis

### OpenNumismat
**[Télécharger OpenNumismat](https://opennumismat.github.io/)** - Logiciel gratuit et open source de gestion de collection de monnaies pour Windows, macOS et Linux. OpenNumismat fournit une base de données puissante pour cataloguer vos pièces, et NumiSync Wizard l'enrichit en renseignant automatiquement les données du catalogue et les prix.

### Numista
**[Visiter Numista.com](https://www.numista.com/)** - Le plus grand catalogue numismatique en ligne au monde, avec des informations détaillées sur les monnaies du monde entier. NumiSync Wizard utilise l'API Numista pour récupérer les données du catalogue, les prix et les images. Vous aurez besoin d'un compte Numista gratuit pour obtenir une clé API.
