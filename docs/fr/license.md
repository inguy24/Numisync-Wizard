## Activer votre licence

### Étape 1 : Copier votre clé de licence

Après l'achat, vous recevrez un e-mail de Polar.sh contenant votre clé de licence. Elle ressemblera à :

```
V1-XXXX-XXXX-XXXX-XXXX
```

### Étape 2 : Ouvrir NumiSync Wizard

Lancez l'application NumiSync Wizard sur votre ordinateur.

### Étape 3 : Entrer la clé de licence

1. Cliquez sur **Settings** dans la barre de menu (ou appuyez sur `Ctrl+,` / `Cmd+,`)
2. Naviguez vers l'onglet **License**
3. Collez votre clé de licence dans le champ de texte
4. Cliquez sur **Activate**

### Étape 4 : Vérifier l'activation

Une fois activée, vous verrez :
- Statut de la licence : **Active**
- Fonctionnalités premium déverrouillées immédiatement
- Plus d'invites intempestives

## Foire aux questions

### Puis-je utiliser ma licence sur plusieurs ordinateurs ?

Oui ! Vous pouvez activer votre licence sur jusqu'à **5 appareils**. Parfait pour la maison, le travail, le portable et les machines de sauvegarde.

### Est-ce un abonnement ?

Non ! **Achat unique, utilisation à vie.** Votre licence n'expire jamais et inclut toutes les fonctionnalités de la version 1.x.

### Qu'en est-il des futures versions majeures (v2.0, v3.0) ?

Votre licence v1 déverrouille définitivement toutes les fonctionnalités v1.x. Les futures versions majeures (v2.0+) avec de nouvelles fonctionnalités significatives peuvent nécessiter une licence séparée, mais les fonctionnalités v1 existantes continueront de fonctionner. Les premiers supporters recevront des remises de mise à niveau.

### J'ai perdu ma clé de licence. Puis-je la récupérer ?

Oui ! Votre clé de licence est liée à votre adresse e-mail. Vérifiez votre e-mail de confirmation d'achat de Polar.sh, ou contactez le support avec votre e-mail d'achat.

### Est-ce que cela soutient le développeur ?

Absolument ! Vos 10 $ contribuent à couvrir :
- L'hébergement du domaine et les certificats SSL
- L'infrastructure API et les serveurs de cache
- Le temps de développement pour de nouvelles fonctionnalités
- Les certificats de signature de code (lorsque disponibles)
- Le café pour les sessions de débogage nocturnes

Chaque licence supporter finance directement le développement continu de NumiSync et le maintient gratuit pour l'utilisation de base.
