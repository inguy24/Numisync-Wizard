## 关于

NumiSync Wizard 将您的 OpenNumismat 钱币收藏数据库与综合性的 Numista 目录相连接，自动匹配您的钱币并为其补充以下信息：

- 详细的目录信息（铸造量、成分、统治者、设计师）
- 四个品相等级的当前市场价格（UNC、XF、VF、F）
- 每种类型的多个版别（年份、铸币标记、类型）
- 高分辨率图片（正面、反面及边缘）
- 智能匹配，支持模糊搜索与手动覆盖

专为希望减少数据录入、将更多时间用于享受收藏乐趣的钱币爱好者打造。

## 功能特色

### 智能钱币匹配
支持面值规范化、发行方解析及非公历纪年（明治、伊斯兰历、泰国佛历）的模糊搜索。

### 精细的数据控制
精确选择需要同步的数据类型：基本目录信息、版别信息或定价数据。只更新您想要的字段。

### 可视化字段对比
并排对比现有数据与新数据。可单独选取字段更新，也可一次性接受所有变更。

### 高级快速定价模式
批量更新所有已匹配钱币的价格，无需逐枚点击。*（需要支持者许可证）*

### 自动传播类型数据
自动将类型级数据（铸造量、成分、统治者、设计师）应用于收藏中所有匹配的钱币。*（需要支持者许可证）*

### 多机同步 *（支持者版）*
通过共享网络驱动器、Dropbox 或 NAS 在多台计算机之间共享 API 缓存和设置。
API 调用计数在所有设备上正确累积。新设备一键导入设置。

### 智能缓存
持久化 API 缓存减少重复请求，遵守 Numista 速率限制。可使用缓存数据离线工作。

### 高级字段映射
自定义 Numista 45+ 个数据来源如何映射到 OpenNumismat 字段，完全掌控您的数据。

## 快速开始

五个简单步骤即可上手：

1. **安装 NumiSync Wizard** 并启动应用程序
2. **打开您的 OpenNumismat 收藏**（.db 文件）
3. **设置 → 添加您的 Numista API 密钥**（从 [numista.com](https://www.numista.com/) 免费获取）
4. **选择钱币 → 点击"Search & Enrich"** 查找匹配项
5. **查看匹配结果 → 接受更改 → 完成！**

详细说明请参阅[安装指南](/zh-CN/installation)和[快速入门指南](/zh-CN/quickstart)。

**平台支持：**
- **Windows 10/11** - 完全支持，提供安装程序
- **macOS 10.13+** - 支持 Intel 和 Apple Silicon（M1/M2/M3）
- **Linux** - Debian/Ubuntu（.deb）、Fedora/RHEL（.rpm）、通用版（AppImage）

## 必要软件

### OpenNumismat
**[下载 OpenNumismat](https://opennumismat.github.io/)** - 适用于 Windows、macOS 和 Linux 的免费开源钱币收藏管理软件。OpenNumismat 提供强大的数据库用于整理您的钱币，而 NumiSync Wizard 通过自动填充目录数据和价格信息来增强它。

### Numista
**[访问 Numista.com](https://www.numista.com/)** - 全球最大的在线钱币目录，收录了来自世界各地的详细钱币信息。NumiSync Wizard 使用 Numista API 获取目录数据、价格和图片。您需要一个免费的 Numista 账户来获取 API 密钥。
