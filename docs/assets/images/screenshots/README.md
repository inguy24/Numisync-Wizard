# Screenshots Directory

This directory contains screenshots for the NumiSync Wizard GitHub Pages site.

## Required Files

Place the following screenshot PNG files here:

1. **main-window.png**
   - Main window with coin collection loaded
   - Shows enriched coins, columns, preview pane
   - ~1600px wide recommended

2. **match-selection.png**
   - Match selection screen showing search results
   - Demonstrates intelligent matching and fuzzy search
   - Multiple potential matches visible

3. **field-comparison.png**
   - Field comparison panel (side-by-side view)
   - Left: existing data, Right: Numista data
   - Shows checkboxes and highlighted differences

4. **issue-picker.png**
   - Issue Picker dialog with multiple variants
   - Shows year, mint mark, type variations
   - 3-5 issues visible

5. **fast-pricing-mode.png**
   - Fast Pricing Mode progress indicator
   - Shows batch update in progress
   - OR: Premium feature upgrade prompt

6. **settings-data.png**
   - Settings panel â†’ Data Settings tab
   - Shows data category checkboxes (Basic, Issue, Pricing, Images)
   - Field mapping section visible

7. **field-mapping.png**
   - Advanced field mapping interface
   - Shows 45+ Numista fields mapped to OpenNumismat fields
   - Customization options visible

## Image Specifications

- **Format:** PNG
- **Width:** 1600px minimum (for high-DPI displays)
- **Compression:** Optimize for web (use tools like TinyPNG if needed)
- **Privacy:** No sensitive data (API keys, license keys, personal info, file paths)

## Updating Screenshots

When updating screenshots for a new version:
1. Replace PNG files with same filenames
2. Ensure new screenshots match the specifications above
3. No changes to markdown files needed (paths remain the same)
