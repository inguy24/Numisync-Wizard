## ライセンスを有効化する

### ステップ1：ライセンスキーをコピーする

購入後、Polar.shからライセンスキーが記載されたメールが届きます。以下のような形式になります：

```
V1-XXXX-XXXX-XXXX-XXXX
```

### ステップ2：NumiSync Wizardを開く

お使いのコンピューターでNumiSync Wizardアプリケーションを起動します。

### ステップ3：ライセンスキーを入力する

1. メニューバーの**Settings**をクリック（または`Ctrl+,` / `Cmd+,`を押す）
2. **License**タブに移動
3. テキストフィールドにライセンスキーを貼り付け
4. **Activate**をクリック

### ステップ4：有効化を確認する

有効化されると、以下が表示されます：
- ライセンスのステータス：**Active**
- プレミアム機能が即座にアンロック
- 通知プロンプトなし

## よくある質問

### 複数のコンピューターで使用できますか？

はい！ライセンスは最大**5台のデバイス**で有効化できます。自宅、職場、ノートパソコン、バックアップマシンに最適です。

### これはサブスクリプションですか？

いいえ！**一度購入すれば永続利用可能です。** ライセンスは有効期限なし、バージョン1.xのすべての機能を含みます。

### 将来のメジャーバージョン（v2.0、v3.0）はどうなりますか？

v1ライセンスはv1.xのすべての機能を永続的にアンロックします。重要な新機能を含む将来のメジャーバージョン（v2.0以降）は別のライセンスが必要になる可能性がありますが、既存のv1機能は引き続き動作します。早期サポーターにはアップグレード割引が提供されます。

### ライセンスキーを紛失した場合、取得できますか？

はい！ライセンスキーはメールアドレスに紐付けられています。Polar.shからの購入確認メールを確認するか、購入時のメールアドレスを添えてサポートにお問い合わせください。

### これは開発者を支援しますか？

もちろんです！皆さまの$10は以下のコストをカバーします：
- ドメインホスティングとSSL証明書
- APIインフラとキャッシュサーバー
- 新機能の開発時間
- コード署名証明書（利用可能な場合）
- 深夜のデバッグセッションのためのコーヒー ☕

すべてのSupporter Licenseは、NumiSyncの継続的な開発を直接支援し、基本的な使用を無料で維持することを可能にします。
